Optimized CubeSat System

Designed and developed a compact CubeSat platform with a 60% volume reduction compared to traditional architectures.

Key Features

  • Modular sensor system (temperature, pressure, humidity)
  • Stackable CAD architecture
  • ANSYS-based structural validation
  • Real-time telemetry

Engineering Work

The system was designed in SolidWorks and validated using ANSYS simulations for both structural integrity and thermal performance. Electronics were integrated using custom perfboards with optimized packaging.

Outcome

Achieved stable descent, reliable data transmission, and efficient spatial utilization.
